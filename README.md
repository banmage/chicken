# 小鸡过河

一个基于 LayaAir 的 HTML5 小游戏。玩家控制小鸡从底部河岸出发，借助移动木船穿过 6 条河道，避开鳄鱼，到达顶部河岸过关。

## 当前版本特性

- 6 条河道采用更容易区分的分层配色
- 鳄鱼障碍物使用侧向造型，并带有张嘴游动动画
- 支持暂停、失败重开、继续下一关
- 支持本地存储最高关卡和星级结果
- 内置隐藏调试快捷键，可快速跳关

## 操作说明

- `←` / `A`：向左移动
- `→` / `D`：向右移动
- `↑` / `W`：向上跳
- `↓` / `S`：向下跳
- `P`：暂停

## 隐藏调试快捷键

- `Alt+1` 到 `Alt+9`：直接进入对应关卡
- `Alt+0`：直接进入第 10 关

这个入口不会显示在游戏 UI 中，主要用于本地调试和快速验证关卡。

## 本地运行

1. 安装依赖

```bash
npm install
```

2. 编译

```bash
npm run build
```

3. 启动静态服务器

```bash
python3 -m http.server 8080
```

4. 浏览器访问

```text
http://localhost:8080
```

## 项目结构

当前项目已经整理为单入口版本，实际运行只依赖以下文件：

```text
chicken/
├── index.html         # 页面入口，加载 LayaAir 和 bin/bundle.js
├── libs/              # LayaAir 运行库
├── src/
│   ├── bundle.ts      # 游戏主逻辑源码
│   ├── config/
│   │   └── Levels.json
│   └── laya.d.ts      # 当前项目使用的简化类型声明
├── bin/
│   ├── bundle.js      # 编译输出
│   └── bundle.js.map
├── package.json
└── tsconfig.json
```

## 构建说明

- TypeScript 编译入口是 `src/bundle.ts`
- 编译输出文件是 `bin/bundle.js`
- `index.html` 实际加载的是 `bin/bundle.js`

如果你修改了游戏逻辑但页面没有变化，先确认是否已经执行过：

```bash
npm run build
```

然后在浏览器里强制刷新一次。

## 玩法规则

1. 小鸡从底部河岸出发
2. 只能通过跳到移动木船上穿越河道
3. 落水或碰到鳄鱼会失败
4. 到达顶部河岸即可过关

## 关卡说明

- 当前共有 10 个关卡
- 难度会随着关卡提升而增加
- 后期关卡会出现更多木船、更快的速度和更多鳄鱼
