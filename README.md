# 小鸡过河 (Chicken Crossing River)

一款基于 LayaAir 3.3.11 引擎开发的 HTML5 小游戏。

## 游戏玩法

帮助小鸡安全渡过河流，到达对岸的鸡窝。

### 操作说明

- **↑ 上键**: 向上跳跃到上一条河道
- **↓ 下键**: 向下跳跃到下一条河道
- **← 左键**: 向左移动
- **→ 右键**: 向右移动
- **P 键**: 暂停游戏

### 游戏规则

1. 小鸡从底部河岸出发
2. 跳过一条条河流，每条河上有移动的小船
3. 小鸡必须跳到船上才能安全过河
4. 避开河流中的鳄鱼障碍物
5. 成功到达顶部河岸即为过关

## 技术栈

- **引擎**: LayaAir 3.3.11
- **语言**: TypeScript
- **打包**: 单文件 HTML 打包

## 项目结构

```
chicken/
├── index.html          # 游戏入口页面
├── src/                # TypeScript 源代码
│   ├── bundle.ts       # 游戏主逻辑
│   └── laya.d.ts       # LayaAir 类型声明
├── libs/               # LayaAir 引擎库
├── bin/                # 编译输出目录
├── tsconfig.json       # TypeScript 配置
└── package.json        # npm 配置
```

## 本地运行

1. 安装依赖：

```bash
npm install
```

2. 编译 TypeScript：

```bash
npx tsc
```

3. 启动本地服务器：

```bash
python3 -m http.server 8080
```

4. 打开浏览器访问：`http://localhost:8080`
